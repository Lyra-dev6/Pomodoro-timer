# Pomodoro-timer
A simple and interactive focus timer built to improve productivity usng the pomodoro technique.

# Overview
This project is my first beginner-friendly web application that allows users to manage focused work sessions and breaks. It was built as a first step into JavaScript developement, focusing on core  concepts like timing functions and user interactions.

# Features
- Pomodoro mode(25 minutes)
- Short break(5 minutes)
- Long break(15 minutes)
- Start, pause and break buttons
- Real-time countdown display
- Notification when a session ends

# Tech Stack
-HTML (structure)
-CSS (Styling)
-JavaScript (Functionality)

# What I learned
- How to use ('setInterval') for time-based functions.
- Handling user input through event listeners
- Structuring a small but complete web application
